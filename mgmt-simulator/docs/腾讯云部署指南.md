# 腾讯云部署指南

## 第一步：购买服务器

1. 打开 cloud.tencent.com，用手机号注册/登录
2. 顶部搜索「轻量应用服务器」
3. 配置选择：
   - 地域：选离你近的（上海/广州）
   - 系统：**Ubuntu 22.04 LTS**
   - 套餐：2核2G / 60GB / 4Mbps（约 45 元/月，新用户可能有优惠）
4. 购买完成后进入「控制台 → 轻量应用服务器」

---

## 第二步：开放防火墙端口

1. 点进你的服务器 → 「防火墙」
2. 点「添加规则」：
   - 应用类型：HTTP(80)
   - 来源：所有 IPv4
3. 再「添加规则」：
   - 应用类型：自定义
   - 协议：TCP
   - 端口：5000
   - 来源：所有 IPv4
4. 保存

---

## 第三步：连接服务器，上传文件

### 方式 A：用腾讯云自带的网页终端（最简单，不用装软件）

控制台 → 服务器详情页 → 右上角「登录」→ 选「使用 OrcaTerm 登录」

### 方式 B：用本机终端 SSH（稍复杂但更方便）

在你的 Windows 终端运行：
```bash
ssh root@你的服务器IP
```
输入购买时设置的密码。

### 上传文件

在你的 Windows 终端（不是服务器终端）运行以下命令，把 bot 文件夹上传到服务器：

```bash
scp -r "C:/Users/yorro/Desktop/claude/bot" root@你的服务器IP:/opt/mgmt-bot
```

---

## 第四步：在服务器上安装环境、运行

连接到服务器终端后，依次执行：

```bash
# 更新系统、安装 Python
apt update && apt install -y python3 python3-pip python3-venv

# 进入项目目录
cd /opt/mgmt-bot

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 测试能不能跑起来（看到 "Web server starting" 就成功了，Ctrl+C 停止）
python bot.py
```

---

## 第五步：设置为系统服务（开机自启、崩了自动重启）

```bash
# 复制服务配置文件
cp /opt/mgmt-bot/mgmt-bot.service /etc/systemd/system/

# 重新加载、启动服务
systemctl daemon-reload
systemctl enable mgmt-bot
systemctl start mgmt-bot

# 查看状态，看到 active (running) 就成功了
systemctl status mgmt-bot
```

---

## 第六步：访问

浏览器或手机打开：
```
http://你的服务器IP:5000
```

看到聊天界面就部署成功了。

---

## 可选：绑定域名 + HTTPS

如果你有域名（或想买一个，.cn 域名约 20 元/年）：

1. 腾讯云买域名 → 解析到服务器 IP（A 记录）
2. 服务器上安装 nginx + certbot 配置 HTTPS

这样可以用 `https://你的域名` 访问，更专业，以后分享给别人也方便。需要的话可以告诉我，我帮你写 nginx 配置。

## 常用运维命令

```bash
# 查看实时日志
tail -f /opt/mgmt-bot/bot.log

# 重启服务
systemctl restart mgmt-bot

# 停止服务
systemctl stop mgmt-bot

# 更新代码后重启
scp -r "C:/Users/yorro/Desktop/claude/bot" root@服务器IP:/opt/mgmt-bot
systemctl restart mgmt-bot
```

## 文件清单

部署到服务器的文件：
- `bot.py` - 后端主程序
- `web/index.html` - 前端页面
- `cards.json` - 卡牌数据（120张：识局×30、读人×29、防身×30、应对×31）
- `requirements.txt` - Python 依赖
- `mgmt-bot.service` - 系统服务配置
